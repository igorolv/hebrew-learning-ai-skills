# Формат выходного файла extracted.md

## Общая структура

```
# Извлечение из урока N, часть M
## שיעור N חלק M

Всего слайдов: X

### Структура презентации
- Слайд 1: категория
- Слайд 2: категория
...

---

## Слайд 1 — 🖼️ Задание по картинке
[описание картинки и различий — заполняется Claude]

---

## Слайд 2 — ✏️ Упражнение: перевод
[содержимое слайда]

---
...
```

## Категории слайдов

### Справочные / грамматические

| Категория | Эмодзи | Описание |
|---|---|---|
| grammar | 📖 | Объяснение грамматики на русском с примерами на иврите |
| vocabulary | 📝 | Таблица новых слов (מילים חדשות) |
| conjugation | 📊 | Таблица спряжения глаголов |
| verb_summary | 📊 | Сводная таблица глаголов по гзарот |
| shem_peula | 📊 | Отглагольные существительные (שם פעולה) |
| numerals | 🔢 | Таблицы числительных (שם מספר): от 10 до 90000 |
| prepositions | 📖 | Таблицы склонения предлогов (מילות יחס) с местоименными суффиксами |

### Упражнения

| Категория | Эмодзи | Описание |
|---|---|---|
| exercise_fill | ✏️ | Вставить слово в пропуск (___) |
| exercise_translate | ✏️ | Перевод предложений (с русского на иврит) |
| exercise_table | ✏️ | Заполнить таблицу (пустые ячейки) |
| exercise_choose | ✏️ | Выбрать правильное слово из вариантов в скобках |
| exercise_oged | ✏️ | Упражнение на אוגד (связка/копула) |
| exercise_morphology | ✏️ | Морфологический разбор: определить שורש, בניין, גזרה по форме глагола |
| exercise_transform | ✏️ | Трансформация: переписать предложения в другом времени (без пропусков) |
| exercise_writing | ✏️ | Сочинение (חיבור): написать текст на заданную тему |
| exercise_questions | ✏️ | Вопросы к тексту: ответить на вопросы по прочитанному |

### Тексты и прочее

| Категория | Эмодзи | Описание |
|---|---|---|
| reading | 📖 | Текст для чтения / диалог / рассказ |
| image_task | 🖼️ | Задание по картинке (найти различия и т.п.) |
| joke | 😄 | Анекдоты и юмористические тексты |
| other | 📄 | Неклассифицированный слайд |

## Маркеры для Claude

В extracted.md встречаются HTML-комментарии-маркеры, которые указывают,
где Claude должен вмешаться:

```html
<!-- CLAUDE: Опиши изображение и найди различия между двумя частями картинки -->
```

Эти маркеры используются ТОЛЬКО для задач, требующих зрения (анализ картинок).
Весь остальной контент извлекается скриптом механически.
