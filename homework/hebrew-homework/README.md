# hebrew-homework

Claude Skill для выполнения домашних заданий по ивриту на основе разобранного урока и обогащённой грамматической шпаргалки.

## Что делает

- Принимает результат работы `hebrew-lesson-extract`
- Использует шпаргалку, созданную `hebrew-lesson-enrich`
- Выполняет упражнения по выбранным слайдам
- Поддерживает основные типы заданий: вставка слов и глаголов, перевод, выбор варианта, таблицы, классификация, тексты для чтения
- Формирует итоговый markdown-файл, готовый для конвертации в DOCX

## Как использовать

### В Claude.ai (Projects)
Добавьте файл `SKILL.md` в Project Knowledge вашего проекта.

### В Claude Code
Скопируйте папку в директорию skills и подключите через конфигурацию.

## Структура

```text
hebrew-homework/
├── SKILL.md                       — основная инструкция
├── README.md
└── references/
    ├── exercise-strategies.md     — стратегии решения упражнений
    ├── formatting-rules.md        — правила оформления выходного markdown
    └── nikud-rules.md             — правила расстановки огласовок
```

## Входные данные

1. `урок_N_часть_M_extracted.md` — результат работы `hebrew-lesson-extract`
2. `Шпаргалка_по_N_уроку_M_часть.md` — результат работы `hebrew-lesson-enrich`
3. Номера слайдов или страниц с заданиями, которые нужно выполнить

## Выходной файл

Markdown-файл с выполненным домашним заданием.

Типичные имена:
- `ДЗ_урок_13_часть_3.md`
- `ДЗ_урок_13_часть_3_слайды_25_26.md`

## Часть пайплайна

Этот skill — третий этап в пайплайне обработки домашних заданий:

1. [hebrew-lesson-extract](../hebrew-lesson-extract) — извлечение структуры из PPTX
2. [hebrew-lesson-enrich](../hebrew-lesson-enrich) — создание грамматической шпаргалки
3. **hebrew-homework** ← вы здесь
4. [hebrew-generate-docx](../hebrew-generate-docx) — конвертация результата в DOCX
