# hebrew-cheatsheet-docx

Преобразование markdown-шпаргалок (грамматических справочников) по ивриту в форматированный Word-документ (DOCX).

## Место в пайплайне

```
PPTX → [hebrew-lesson-extract] → extracted.md
                                       ↓
                                [hebrew-lesson-enrich] → Шпаргалка.md
                                                              ↓
                                                    [hebrew-cheatsheet-docx] → Шпаргалка.docx
```

## Вход

Markdown-файл — результат скила `hebrew-lesson-enrich`.

Формат имени: `Шпаргалка_по_{N}_уроку_{M}_часть.md`

## Выход

Форматированный DOCX с:
- Иерархией заголовков (разделы → подразделы → подподразделы)
- Таблицами спряжений и лексики (David 18pt для иврита, Arial 12pt для русского)
- Blockquotes с синей левой границей для ключевых правил
- Маркированными списками
- Контрастными парами (иврит + перевод)
- Разрывами страниц перед крупными разделами
- Корректной поддержкой RTL/BiDi для иврита

## Отличия от hebrew-generate-docx

| | Домашнее задание | Шпаргалка |
|---|---|---|
| Скилл | hebrew-generate-docx | hebrew-cheatsheet-docx |
| Структура | Плоская (слайды) | Иерархическая (разделы) |
| Blockquotes | Нет | Есть |
| Списки | Нумерованные упражнения | Маркированные |
| Разрывы страниц | Нет | Да (между разделами) |
| Иврит в таблицах | 14pt | 18pt |
| Поля страницы | 2.54 см | 2.0 см |

## Зависимости

- Node.js
- npm пакет `docx`
