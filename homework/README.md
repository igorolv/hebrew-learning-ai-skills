# Homework · Hebrew Learning Pipeline

Набор Claude Skills для обработки уроков иврита и домашних заданий: от PPTX-презентации до готового DOCX с выполненными упражнениями.

---

## Что входит в pipeline

Папка `homework/` содержит четыре связанных skill'а:

1. **hebrew-lesson-extract** — извлекает содержимое PPTX-урока в структурированный markdown
2. **hebrew-lesson-enrich** — превращает extracted-файл в подробную грамматическую шпаргалку
3. **hebrew-homework** — выполняет упражнения по выбранным слайдам
4. **hebrew-generate-docx** — собирает результат в Word-документ

---

## Подробно по шагам

### Шаг 1 · hebrew-lesson-extract

**Что делает:** извлекает из PPTX текст, таблицы, изображения и структуру слайдов.

| | |
|---|---|
| **Вход** | PPTX-файл урока иврита |
| **Выход** | `урок_N_часть_M_extracted.md` + `images/` |
| **Назначение** | Получить сырой, но структурированный материал урока |

---

### Шаг 2 · hebrew-lesson-enrich

**Что делает:** строит грамматическую шпаргалку по extracted-файлу.

| | |
|---|---|
| **Вход** | `урок_N_часть_M_extracted.md` |
| **Выход** | `Шпаргалка_по_N_уроку_M_часть.md` |
| **Назначение** | Подготовить справочный материал для выполнения упражнений |

---

### Шаг 3 · hebrew-homework

**Что делает:** выполняет домашнее задание по указанным слайдам урока.

| | |
|---|---|
| **Вход** | extracted-файл + шпаргалка + номера слайдов |
| **Выход** | `ДЗ_урок_N_часть_M*.md` |
| **Назначение** | Получить оформленное выполненное домашнее задание |

---

### Шаг 4 · hebrew-generate-docx

**Что делает:** конвертирует выполненное домашнее задание из markdown в DOCX.

| | |
|---|---|
| **Вход** | markdown-файл от `hebrew-homework` |
| **Выход** | DOCX-файл с типографикой для иврита и русского |
| **Назначение** | Подготовить задание для печати, отправки или архива |

---

## Конвенции именования файлов

| Тип файла | Формат имени | Пример |
|---|---|---|
| Извлечённый урок | `урок_N_часть_M_extracted.md` | `урок_13_часть_3_extracted.md` |
| Шпаргалка | `Шпаргалка_по_N_уроку_M_часть.md` | `Шпаргалка_по_13_уроку_3_часть.md` |
| Домашнее задание | `ДЗ_урок_N_часть_M.md` | `ДЗ_урок_13_часть_3.md` |
| Домашнее задание по слайдам | `ДЗ_урок_N_часть_M_слайды_X_Y.md` | `ДЗ_урок_13_часть_3_слайды_25_26.md` |
| Итоговый DOCX | то же имя, но с `.docx` | `ДЗ_урок_13_часть_3_слайды_25_26.docx` |
