# Homework Pipeline

Формальное описание pipeline для AI-агентов. Человекочитаемое описание — в [README.md](README.md).

## Что такое pipeline

Pipeline — упорядоченный граф скилов, где выход одного скила является входом следующего. Скилы выполняются последовательно; между шагами AI показывает результат пользователю и ждёт подтверждения.

---

## Граф

```
                PPTX урока (от пользователя)
                         |
                         v
              [hebrew-lesson-extract]
                         |
                         | урок_N_часть_M_extracted.zip
                         |   (md + images/)
                         v
              [hebrew-lesson-enrich]
                         |
                         | Шпаргалка_по_{N}_уроку_{M}_часть.md
                         |
              +----------+----------+
              |                     |
              v                     v
   [hebrew-homework-solve]   [hebrew-lesson-docx]
              |                     |
              | ДЗ_урок_N_часть_M   | Шпаргалка.docx
              | _слайды_*.md        |
              v
   [hebrew-homework-docx]
              |
              | ДЗ.docx
```

## Ветвления

Pipeline ветвится после `hebrew-lesson-enrich`:

- **Ветка ДЗ**: enrich -> homework-solve -> homework-docx
- **Ветка шпаргалки**: enrich -> lesson-docx

Ветки независимы — можно запускать в любом порядке или только одну из них.

---

## Стыки: что передаётся между скилами

### extract -> enrich

| | |
|---|---|
| **Артефакт** | `урок_N_часть_M_extracted.zip` |
| **Содержимое** | `урок_N_часть_M_extracted.md` + `images/` |
| **Формат md** | Структурированный markdown с классифицированными слайдами (grammar, vocabulary, exercise_*, reading, joke, image_task) |

### enrich -> homework-solve

| | |
|---|---|
| **Артефакт** | `Шпаргалка_по_{N}_уроку_{M}_часть.md` |
| **Что использует solve** | Таблицы спряжения, списки новых слов, грамматические правила — для правильного выполнения упражнений |

### enrich -> lesson-docx

| | |
|---|---|
| **Артефакт** | `Шпаргалка_по_{N}_уроку_{M}_часть.md` |
| **Что использует docx** | Весь файл целиком — конвертирует markdown в Word |

### extract -> homework-solve

| | |
|---|---|
| **Артефакт** | `урок_N_часть_M_extracted.md` (из ZIP) |
| **Что использует solve** | Тексты упражнений, инструкции к заданиям, изображения (для picture-diff) |

### homework-solve -> homework-docx

| | |
|---|---|
| **Артефакт** | `ДЗ_урок_{N}_часть_{M}_слайды_{СЛАЙДЫ}.md` |
| **Что использует docx** | Весь файл целиком — конвертирует markdown в Word |

## Данные от пользователя vs между скилами

### От пользователя (внешние входы)

| Что | На каком шаге | Обязательно |
|-----|--------------|-------------|
| PPTX-файл урока | extract | да |
| Номера слайдов с заданиями | homework-solve | да, всегда спрашивать |
| Предыдущие шпаргалки (для частей 2+) | enrich | нет, но рекомендуется |

### Между скилами (внутренние артефакты)

| Артефакт | Откуда | Куда |
|----------|--------|------|
| `урок_N_часть_M_extracted.zip` | extract | enrich, homework-solve |
| `Шпаргалка_по_{N}_уроку_{M}_часть.md` | enrich | homework-solve, lesson-docx |
| `ДЗ_урок_{N}_часть_{M}_слайды_{СЛАЙДЫ}.md` | homework-solve | homework-docx |

---

## Порядок запуска

### Полный pipeline (ДЗ + шпаргалка)

```
1. extract      — обязательно первый
2. enrich       — после extract
3. homework-solve + lesson-docx — после enrich, параллельно или в любом порядке
4. homework-docx — после homework-solve
```

### Только ДЗ

```
1. extract
2. enrich
3. homework-solve
4. homework-docx
```

### Только шпаргалка

```
1. extract
2. enrich
3. lesson-docx
```

---

## Что делать, если предыдущий скил не выполнен

Каждый скил проверяет наличие входных данных. Если артефакт предыдущего скила не найден:

1. Сообщить пользователю, какой артефакт отсутствует
2. Предложить запустить предыдущий скил
3. Не пытаться выполнить текущий скил без входных данных

## Конвенции именования

| Артефакт | Формат имени |
|----------|-------------|
| Извлечённый урок (ZIP) | `урок_{N}_часть_{M}_extracted.zip` |
| Извлечённый урок (md) | `урок_{N}_часть_{M}_extracted.md` |
| Шпаргалка | `Шпаргалка_по_{N}_уроку_{M}_часть.md` |
| Домашнее задание | `ДЗ_урок_{N}_часть_{M}_слайды_{СЛАЙДЫ}.md` |
| DOCX (ДЗ или шпаргалка) | то же имя, `.md` -> `.docx` |

Подробности по суффиксу `{СЛАЙДЫ}` — в SKILL.md скила `hebrew-homework-solve`.
