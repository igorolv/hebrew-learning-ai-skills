# Правила оформления markdown для домашних заданий

## Общая структура файла

```markdown
# Домашнее задание — Урок N, часть M

---

## Слайд X — Упражнение: вставить глагол (הווה)

[оригинальная инструкция на иврите]

| № | Текст |
|---|---|
| 1 | [предложение с вставленным словом] |
|   | [перевод] |
| 2 | [предложение с вставленным словом] |
|   | [перевод] |

---

## Слайд Y — Упражнение: перевод (рус→ивр)

[оригинальная инструкция на иврите]

| № | Текст |
|---|---|
| 1 | [перевод на иврит] |
|   | [исходное предложение на русском] |

---
```

## Формат таблиц упражнений

Все упражнения со списками (fill-in, translation, choice, copula) оформляются как markdown-таблицы с двумя колонками:

| Колонка | Содержимое |
|---------|------------|
| **№** | Номер пункта — только в первой строке пункта. Строки продолжения — пустая ячейка |
| **Текст** | Иврит, перевод, объяснение — каждый на отдельной строке таблицы |

Количество строк на пункт зависит от типа упражнения:
- fill-in, translation, copula — **2 строки** (иврит + перевод)
- choice — **3 строки** (иврит + объяснение + перевод)

## Правила для разных типов упражнений

### Упражнение «вставить слово» (fill-in)

- Вставленное слово выделить **жирным**: `**מוֹסִיף**`
- Остальной текст — огласовки по общим правилам nikud-rules (новые слова, глаголы из таблиц, имена собственные и т.д.)
- Инфинитив глагола в скобках оставить как в оригинале
- Каждый пункт — 2 строки таблицы: иврит + перевод

**Пример:**
```
| № | Текст |
|---|---|
| 1 | אני **מוֹסִיף** כל יום מילה חדשה לַמַּחְבֶּרֶת שלי כדי לְהַעֲשִׁיר את אוֹצַר הַמִּילִים שלי. (להוסיף) |
|   | Я добавляю каждый день новое слово в свою тетрадь, чтобы обогатить свой словарный запас. |
| 2 | הַזַּמֶּרֶת **מוֹפִיעָה** הערב בַּפֶּסְטִיבָל גדול מול אלפי אנשים שמחכים לה בשמחה. (להופיע) |
|   | Певица выступает сегодня вечером на большом фестивале перед тысячами людей, которые ждут её с радостью. |
```

### Упражнение «перевод с русского на иврит»

- Перевод на иврит — полноценное предложение
- Исходное русское предложение — курсивом `*текст*`
- Новые слова урока и глаголы из таблиц — с огласовками
- Каждый пункт — 2 строки таблицы: иврит + русский оригинал курсивом

**Пример:**
```
| № | Текст |
|---|---|
| 1 | אני מוֹסִיפָה סוּכָּר לקפה כל בוקר, אחרת הוא מַר מדי. |
|   | *Я добавляю сахар в кофе каждое утро, иначе он слишком горький.* |
| 2 | הוֹסַפְנוּ עוד כמה פרטים למכתב, כדי שייראה יותר מקצועי. |
|   | *Мы добавили ещё несколько деталей в письмо, чтобы оно выглядело профессиональнее.* |
```

### Упражнение «перевод с иврита на русский»

- Огласовки в ивритском предложении — по общим правилам nikud-rules
- Перевод близкий к тексту
- Каждый пункт — 2 строки таблицы: иврит + перевод

### Упражнение «таблица спряжения»

Без изменений — уже таблица. Воспроизвести в стандартном markdown.

- Все вписанные формы — **жирным** и с огласовками
- Оригинальные данные (которые были даны) — без выделения

**Пример:**
```
| עבר | הווה | עתיד |
| --- | --- | --- |
| הוֹפִיעַ (הוא) | | |
| | היא **מוֹשִׁיבָה** | |
| | | **נוֹדִיעַ** (אנחנו) |
```

### Упражнение «выбор слова»

- Каждый пункт — 3 строки таблицы: иврит с вариантами, строка «Выбрано: ...», перевод

**Пример:**
```
| № | Текст |
|---|---|
| 1 | האריה רצה, שהזאב **יְחַלֵּק** (חילק, יחלק, לחלק) את השָׁלָל. |
|   | Выбрано: **יחלק** — после ש + רצה требуется будущее время (сослагательное значение) |
|   | Лев хотел, чтобы волк разделил добычу. |
| 2 | החיילים יצאו ל**מַסָּע** (טיול, דרך, מסע) ארוך וקשה. |
|   | Выбрано: **מסע** — «поход, экспедиция» подходит к контексту солдат и тяжёлого пути |
|   | Солдаты отправились в долгий и тяжёлый поход. |
```

### Упражнение «אוגד» (связка)

- Каждый пункт — 2 строки таблицы: иврит + перевод

Правило выбора связки:
- Подлежащее м.р. ед.ч. → **הוא**
- Подлежащее ж.р. ед.ч. → **היא**
- Подлежащее м.р. мн.ч. → **הם**
- Подлежащее ж.р. мн.ч. → **הן**

**Пример:**
```
| № | Текст |
|---|---|
| 1 | הַסֵּפֶר הזה **הוא** מעניין מאוד. |
|   | Эта книга — очень интересная. |
| 2 | הַתַּעֲרוּכָה **היא** חשובה לְאָמָּנוּת הַיִּשְׂרְאֵלִית. |
|   | Выставка важна для израильского искусства. |
```

### Упражнение с текстом / анекдотами

Без изменений — не таблица, а связный текст.

- Если задание — выбрать правильный конец анекдота, указать выбранный вариант и объяснить юмор
- Если задание — ответить на вопросы по тексту, дать ответы на иврите с переводом

### Текст для чтения (text-reading)

Без изменений — не таблица, а связный текст (рассказ, история, диалог).

**Структура:**
```
### [Заголовок с огласовками]

[Абзац на иврите с огласовками]

[Перевод этого абзаца на русский]

[Следующий абзац на иврите с огласовками]

[Перевод следующего абзаца на русский]
```

**Правила:**
- Каждый абзац оригинала сохраняется как отдельный абзац
- После абзаца на иврите — пустая строка — абзац перевода на русский — пустая строка
- Перевод близкий к тексту, но литературно читаемый
- Заголовок текста оформлять как `###`
- Если перед текстом есть инструкция (תאזינו, תקראו) — оставить её без перевода
- Диалоги: каждая реплика — отдельная строка, начинается с тире `—`

### Упражнение «классификация» (распределить слова по группам)

Без изменений — не таблица, а списки слов по группам.

- Воспроизвести обе группы
- Для каждого слова — добавить краткий перевод в скобках

### Упражнение «найди отличия» (picture-diff)

- Инструкция перед таблицей — всегда стандартная: `מִצְאוּ הֶבְדֵּלִים בֵּין שְׁתֵּי הַתְּמוּנוֹת`
- Стиль описания: краткое сравнение `למעלה — ...; למטה — ...`
- Каждый пункт — 2 строки таблицы: иврит + перевод
- Огласовки — по общим правилам nikud-rules

**Пример:**
```
| № | Текст |
|---|---|
| 1 | למעלה — כּוֹבַע כָּתוֹם עם סֶרֶט אָדוֹם; למטה — כּוֹבַע קַשׁ צָהוֹב עם סֶרֶט כָּחוֹל. |
|   | Наверху — оранжевая шляпа с красной лентой; внизу — жёлтая соломенная шляпа с голубой лентой. |
| 2 | למעלה — לַיֶּלֶד יש מִשְׁקָפַיִם; למטה — אין מִשְׁקָפַיִם. |
|   | Наверху — у мальчика есть очки; внизу — очков нет. |
```

## Технические правила markdown → docx

1. **Не использовать HTML-теги** — только чистый markdown
2. **Заголовки**: `#` для названия, `##` для слайдов, `###` для подразделов
3. **Горизонтальные разделители** `---` между слайдами
4. **Таблицы упражнений**: двухколоночные таблицы `| № | Текст |` для всех упражнений со списками
5. **Таблицы спряжения**: стандартный markdown с `|` и `---`
6. **Жирный**: `**текст**` для вставленных слов
7. **Курсив**: `*текст*` для русских оригиналов при переводе рус→ивр
8. **Не использовать** нумерованные списки `1. 2. 3.` для упражнений — вместо них таблицы
9. Тексты для чтения, анекдоты, классификация — абзацы и списки без таблиц
