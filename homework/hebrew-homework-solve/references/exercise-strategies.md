# Стратегии решения упражнений по ивриту

## 1. Вставить глагол в нужной форме

### Алгоритм

1. **Прочитать инструкцию**: определить требуемое время (הווה / עבר / עתיד)
2. **Определить инфинитив**: обычно указан в скобках после предложения
3. **Определить биньян и гзару глагола**: найти в сводной таблице глаголов урока или в шпаргалке
4. **Определить подлежащее**: найти в предложении — кто выполняет действие
5. **Определить лицо, род, число**: по подлежащему
6. **Проспрягать**: используя таблицу спряжения из шпаргалки
7. **Вставить форму с огласовками**
8. **Расставить огласовки в остальном тексте** предложения по общим правилам nikud-rules

### Подсказки по определению подлежащего

| Слово/фраза | Лицо | Род | Число |
|---|---|---|---|
| אני | 1 | определяется контекстом | ед. |
| אתה | 2 | м. | ед. |
| את | 2 | ж. | ед. |
| הוא, [имя м.р.] | 3 | м. | ед. |
| היא, [имя ж.р.] | 3 | ж. | ед. |
| אנחנו | 1 | — | мн. |
| אתם | 2 | м. | мн. |
| אתן | 2 | ж. | мн. |
| הם, [сущ. м.р. мн.] | 3 | м. | мн. |
| הן, [сущ. ж.р. мн.] | 3 | ж. | мн. |

### Время по контексту

- **הווה**: כל יום, תמיד, עכשיו, בדרך כלל
- **עבר**: אתמול, לפני, בשנה שעברה, כבר (в прошлом контексте)
- **עתיד**: מחר, בעתיד, בשבוע הבא

## 2. Перевод с русского на иврит

### Алгоритм

1. **Определить подлежащее и сказуемое** в русском предложении
2. **Определить время**: прошедшее / настоящее / будущее
3. **Найти нужный глагол** в списке глаголов урока
4. **Проспрягать глагол** в нужном времени, лице, роде и числе
5. **Построить предложение** по ивритскому порядку слов (обычно SVO)
6. **Использовать лексику урока** — не выдумывать свои слова, если в уроке есть подходящие
7. **Проверить согласование**: глагол с подлежащим, прилагательное с существительным

### Типичные конструкции

- «не ... а ...» → לא ... אלא ...
- «не только ... но и ...» → לא רק ... אלא גם ...
- «несмотря на то что» → אף על פי ש... / למרות ש...
- «тем не менее» → אף על פי כן
- «для того чтобы» → כדי ש... / כדי + инфинитив
- «после того как» → אחרי ש...
- «перед тем как» → לפני ש...

### Особенности перевода

- **Конструкция с את**: прямое дополнение с определённым артиклем требует את
- **Смихут** (сопряжённые конструкции): при переводе «X чего-то» — использовать смихут
- **Предлоги**: не калькировать русские предлоги, использовать правильные ивритские (на кого-то злиться = כועס על)

## 3. Заполнение таблицы спряжения

### Алгоритм

1. Определить глагол и его корень по данным формам
2. Определить биньян и гзару
3. По одной известной форме вычислить все остальные по модели
4. Все вписанные формы — с огласовками

### Контроль: что проверить

- Префикс соответствует гзаре (ma/me/mo для הווה разных гзарот)
- Суффиксы правильные для лица/рода/числа
- Дагеш стоит где нужно (для פ״נ)
- Matres lectionis сохранены

---

## 4. Вставить אוגד (связку)

### Правило

В именных предложениях (без глагола действия) связка — местоимение 3-го лица, согласованное с подлежащим:

| Подлежащее | Связка |
|---|---|
| Ед.ч. м.р. (הספר הזה, הבית) | הוא |
| Ед.ч. ж.р. (התערוכה, העיר) | היא |
| Мн.ч. м.р. (הבתים, החיים) | הם |
| Мн.ч. ж.р. (הבנות, המלחמות) | הן |

**Внимание:**
- Определять род по существительному, а не по определению
- Некоторые слова имеют неочевидный род: חיים — м.р. мн.ч. (הם), שמיים — м.р. мн.ч. (הם)
- Парные существительные (עיניים, ידיים) — ж.р. мн.ч.

---

## 5. Выбрать правильное слово

### Алгоритм

1. Прочитать предложение целиком
2. Определить, что требуется: глагол в правильной форме? правильное значение? правильная часть речи?
3. Проверить согласование выбранного варианта с контекстом
4. Объяснить выбор

---

## 6. Классификация слов

### Алгоритм

1. Для каждого слова определить значение
2. Классифицировать по указанному критерию
3. Добавить перевод для каждого слова
