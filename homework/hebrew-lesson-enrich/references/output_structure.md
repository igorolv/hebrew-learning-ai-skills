# Структура выходного файла шпаргалки

## Общая структура документа

### Заголовок

Title format: `Шпаргалка по N уроку — часть M`
Optionally, the main grammar topic can be added after an em-dash:
`Шпаргалка по 13 уроку — часть 3` or
`Грамматика урока 13 (часть 1) — Биньян הפעיל`

Second-level heading is always the Hebrew lesson identifier: `שיעור N חלק M`

```
# Шпаргалка по N уроку — часть M
## שיעור N חלק M
---

# 1. [Первая грамматическая тема] — подробное объяснение
## 1.1 [Что это такое]
## 1.2 [Правила / морфология]
## 1.3 [Спряжение с полными парадигмами]
## 1.4 [Контрастные примеры из урока]
## 1.5 [Исторический комментарий] (опционально)
---

# 2. [Вторая грамматическая тема] (если есть)
...
---

# N. Сводная таблица глаголов (кумулятивная по всем частям урока)
## N.1 שלמים
```

## Шаблоны разделов

### Грамматическая тема

```markdown
# 1. [Название темы на иврите] — [описание на русском]

## 1.1 Что такое [тема]

[Объяснение на русском: что это за явление, к какой группе относится,
какие корни затрагивает. 2-4 абзаца.]

Примеры исходных глаголов:

יָרַד — спускаться
יָצָא — выходить
...

## 1.2 [Морфологическое правило / Каузативная связь / и т.д.]

[Таблица или описание правила.]

| פעל (действие) | Перевод | הפעיל (побуждение) | Перевод |
|---|---|---|---|
| יָרַד | спускаться | הוֹרִיד | спускать, снижать |
...

## 1.3 Главное правило огласовок

[Описание паттерна огласовок для данной гзары.]

> Выделенные правила оформляются как цитата (blockquote).

## 1.4 Спряжение: [инфинитив] ([перевод])

Корень: **[корень]**

### הווה (настоящее время)
[Таблица: м.р./ж.р. × ед.ч./мн.ч.]

### עבר (прошедшее время)
[Таблица: все лица]

### עתיד (будущее время)
[Таблица: все лица]

## 1.5 Контрастные примеры из урока

[Пары предложений: פעל vs הפעיל]

הַיֶּלֶד יוֹרֵד לָרְחוֹב.
Ребёнок спускается на улицу.

אִמָּא מוֹרִידָה אֶת הַיֶּלֶד לָרְחוֹב.
Мама спускает ребёнка на улицу.

## 1.6 Исторический комментарий

[Этимология, связь с другими семитскими языками, библейские параллели.]
```

```markdown
[та же структура]
```

### Диагностическая памятка

```markdown
# N. Как различать гзарот: сводная памятка

[Вводный абзац: почему различение важно.]

| Характеристика | שלמים | פ״נ | ע״ו/י | פ״י |
|---|---|---|---|---|
| Префикс инфинитива | לְהַ [leha] | לְהַ [leha] | לְהָ [lehā] | לְהוֹ [lehō] |
| Префикс наст. вр. | מַ [ma] | מַ [ma] | מֵ [me] | מוֹ [mo] |
| Префикс прош. вр. | הִ [hi] | הִ [hi] | הֵ [he] | הוֹ [ho] |
| Дагеш | нет | есть (во 2-й букве) | нет | нет |
| Видимых корневых букв | 3 | 2 (+ дагеш) | 2 | 2 |

**Ключевое правило распознавания:**

* Если все префиксы имеют гласный **[o]** — это **גזרת פ״י**.
* Если префиксы **[me / he]** — это **ע״ו/י**.
* Если **[ma / hi]** и есть **дагеш** — это **פ״נ**.
* Если **[ma / hi]** и три видимые корневые буквы — это **שלמים**.
```

### Новые слова

Vocabulary tables always use these exact column headers (standardized in урок 13 format).
Do NOT use `Иврит | Перевод | Примечания` (older format from урок 11).

```markdown
# N. Новые слова урока (часть M)

## N.1 Существительные (שמות עצם)

| С огласовками | Без огласовок | Перевод |
|---|---|---|
| מַעְגָּל | מעגל | круг, цикл |
...

## N.2 Прилагательные (שמות תואר)

| С огласовками | Без огласовок | Перевод |
|---|---|---|
| סוֹעֵר | סוער | бурный, бушующий |
...

## N.3 Глаголы (פעלים)

| С огласовками | Без огласовок | Биньян | Перевод |
|---|---|---|---|
| תָּקַף | תקף | פעל | нападать, атаковать |
...

## N.4 Служебные слова (מיליות)

| С огласовками | Без огласовок | Перевод |
|---|---|---|
| אַף עַל פִּי כֵן | אף על פי כן | тем не менее |

## N.5 Выражения (ביטויים)

| С огласовками | Без огласовок | Перевод |
|---|---|---|
| מְחִיאוֹת כַּפַּיִם | מחיאות כפיים | аплодисменты |
...
```

### Итоговая таблица глаголов с שם פעולה

In this table (and ONLY this table), infinitives are **bold** to visually separate
them from their שם פעולה below. The arrow symbol ↳ marks verbal nouns.

```markdown
# N. Сводная таблица глаголов הפעיל с отглагольными существительными (часть M)

| С огласовками | Без огласовок | Перевод |
|---|---|---|
| **לְהוֹסִיף** | להוסיף | добавлять |
| ↳ הוֹסָפָה | הוספה | добавление |
| **לְהוֹפִיעַ** | להופיע | появляться, выступать |
| ↳ הוֹפָעָה | הופעה | появление, выступление |
...
```

## Форматирование

### Общие правила
- Заголовки: `#` для крупных разделов, `##` для подразделов, `###` для подподразделов
- Разделитель `---` между крупными разделами
- Таблицы: стандартный markdown с `|`
- Цитаты (blockquote `>`) для ключевых правил и формулировок
- Нумерация разделов: сквозная по всему документу

### Иврит и русский
- Примеры на иврите идут первой строкой
- Русский перевод — второй строкой, без пустой строки между ними (или с одной пустой)
- Формат пар: без буллетов, без нумерации, просто текст

### Огласовки
- В таблицах словаря — обязательно
- В парадигмах спряжений — обязательно
- В примерах-предложениях внутри грамматических разделов — добавлять
- В названиях разделов — только если это термин (напр. שם פעולה, הפעיל)

### Что НЕ включать
- Упражнения (exercise_*)
- Тексты для чтения (reading)
- Анекдоты (joke)
- Задания по картинкам (image_task)
- Слайд с оглавлением / заголовком (title slide)
