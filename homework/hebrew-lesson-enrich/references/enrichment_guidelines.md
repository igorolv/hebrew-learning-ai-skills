# Правила лингвистического обогащения

## 1. Обработка грамматических слайдов

### 1.1 Что считается грамматическим слайдом

Слайды с категорией `grammar` в extracted.md содержат объяснения на русском языке
с примерами на иврите. Они — основной источник для грамматических разделов шпаргалки.

Также грамматически значимы:
- `conjugation` — таблицы спряжений → полные парадигмы
- `verb_summary` — сводки глаголов по гзарот → кумулятивная таблица
- `shem_peula` — таблицы שם פעולה → раздел отглагольных существительных

### 1.2 Алгоритм обработки грамматического слайда

1. **Определить тему**: какое грамматическое явление описывается
2. **Извлечь содержание**: правила, примеры, таблицы
3. **Обогатить**:
   - Переформулировать объяснение более структурно и подробно
   - Добавить морфологическую формулу / правило
   - Добавить полную парадигму (если есть conjugation-слайд)
   - Добавить контрастные пары (פעל ↔ הפעיל и т.д.)
   - Добавить диагностический признак (как распознать)
   - Добавить исторический / этимологический комментарий (опционально)
4. **Расставить огласовки** во всех ключевых формах

### 1.3 Что значит «обогатить»

Извлечённый текст из презентации часто представляет собой конспективные заметки
преподавателя. Задача обогащения — превратить их в самодостаточный справочник.

**Пример (извлечённый текст):**
```
Правила спряжения глаголов группы פ״י в породе הפעיל очень просты:
все без исключения префиксы огласованы [o]...
```

**Обогащённый результат:**
```markdown
## 1.3 Главное правило огласовок

Во всех формах глаголов פ״י:

* инфинитив: **לְהוֹ-**
* настоящее время: **מוֹ-**
* прошедшее время: **הוֹ-**
* будущее время: **אוֹ- / תּוֹ- / יוֹ- / נוֹ-**

В остальном спряжение соответствует стандартному биньяну הפעיל.

> Во всех префиксах появляется гласный **[o]** (холам).
> Именно это — главный диагностический признак גזרת פ״י.
```

Обратите внимание: обогащение добавляет структуру, полноту, ясность,
но **не выдумывает информацию**. Все примеры и правила должны быть
лингвистически корректны.

## 2. Правила работы с огласовками (никуд)

### 2.1 Где обязательно ставить огласовки

| Контекст | Огласовки |
|---|---|
| Таблицы словаря (новые слова) | ✅ Обязательно |
| Парадигмы спряжений | ✅ Обязательно |
| שם פעולה в таблицах | ✅ Обязательно |
| Ключевые формы в правилах (инфинитив, пример гзары) | ✅ Обязательно |
| Контрастные примеры (пары предложений) | ✅ Желательно |
| Названия гзарот и биньянов | ✅ При первом упоминании |

### 2.2 Где НЕ ставить огласовки

| Контекст | Огласовки |
|---|---|
| Тексты упражнений | ❌ (но упражнения и не включаются) |
| Тексты для чтения | ❌ (не включаются) |
| Окружающий контекст в предложениях-примерах | По усмотрению |

### 2.3 Критическое правило: сохранение букв

**НИКОГДА не удалять буквы при добавлении огласовок.**

Matres lectionis (אמות קריאה) — буквы ו, י, א, ה — которые в неогласованном
тексте обозначают гласные, должны сохраняться при добавлении огласовок.

**Правильно:** הוֹרִיד (буква י сохранена)
**Неправильно:** הוֹרִד (буква י удалена)

**Правильно:** מוֹרִידָה (буквы ו и י сохранены)
**Неправильно:** מוֹרִדָה (буква י удалена)

### 2.4 Источники для проверки огласовок

1. Огласованные формы из conjugation-слайдов (если есть)
2. Огласованные формы из shem_peula-слайдов
3. Огласованные формы из vocabulary-слайда
4. Собственное знание морфологии иврита (если 1-3 недоступны)
5. Предыдущие шпаргалки того же урока (для единообразия)

## 3. Обработка словаря новых слов

### 3.1 Исходный формат (из extracted.md)

Слайд `vocabulary` обычно содержит одну общую таблицу с колонками:
שם פועל | שם תואר | פעלים | מיליות | ביטויים

Слова даны **без огласовок** и **без переводов**.

### 3.2 Целевой формат

Разбить на отдельные подтаблицы по категориям. Для каждого слова:

1. **Добавить огласовки** — на основе знания морфологии
2. **Добавить перевод** — точный, не приблизительный
3. **Для глаголов — указать биньян**

### 3.3 Категории

| Категория | Заголовок раздела | Заголовок на иврите |
|---|---|---|
| Существительные | Существительные | שמות עצם |
| Прилагательные | Прилагательные | שמות תואר |
| Глаголы | Глаголы | פעלים |
| Служебные слова | Служебные слова | מיליות |
| Выражения | Выражения | ביטויים |

**Важно:** Если категория пуста (нет слов данного типа в уроке),
соответствующий подраздел **не создаётся** — пустых таблиц быть не должно.

### 3.4 Определение биньяна для глаголов

Глаголы в vocabulary-слайде даны без огласовок. Для указания биньяна:

1. Проверить, есть ли глагол в verb_summary-слайде (там обычно сгруппированы по גזרות)
2. Определить биньян по морфологическим признакам:
   - Префикс ה → הפעיל или התפעל
   - Удвоение средней буквы → פיעל
   - Префикс נ → נפעל
   - Префикс הסת / הת → התפעל
   - Простая трёхбуквенная основа → פעל

### 3.5 Перевод

- Дать **основное значение** (1-3 слова)
- При необходимости — уточнение в скобках
- Порядок: наиболее частое значение первым

**Примеры:**
- תָּקַף → нападать, атаковать
- שִׁינָּה → менять, изменять
- קֶרֶן → фонд; луч; рог

## 4. Построение кумулятивной таблицы глаголов

### 4.1 Принцип кумулятивности

Если урок состоит из нескольких частей (напр., урок 13 = 3 части),
каждая последующая шпаргалка включает **все** глаголы הפעיל из всех
предыдущих частей + новые из текущей.

Для части 1: только глаголы части 1
Для части 2: глаголы частей 1 + 2
Для части 3: глаголы частей 1 + 2 + 3

### 4.2 Источники

- verb_summary-слайд текущей части
- verb_summary из предыдущих шпаргалок (через project_knowledge_search)
- conjugation-слайды (могут содержать глаголы, не перечисленные в сводке)

### 4.3 Организация по гзарот

Порядок гзарот в таблице:
1. שלמים (полные) — самая большая группа
2. גזרת פ״נ
3. גזרת ע״ו/י
4. גזרת פ״י

### 4.4 Формат записи

| Инфинитив с огласовками | Без огласовок | Перевод |
|---|---|---|
| לְהַרְגִּישׁ | להרגיש | чувствовать |

## 5. Обработка שם פעולה

### 5.1 Модели образования по гзарот

| Гзара | Модель | Пример |
|---|---|---|
| שלמים | הַ_ְ_ָ_ָה | הַרְגָּשָׁה |
| פ״נ | הַ_ָּ_ָה (с дагешем) | הַכָּרָה |
| ע״ו/י | הֲ_ָ_ָה (с хатаф-патах) | הֲבָנָה |
| פ״י | הוֹ_ָ_ָה (с холам) | הוֹרָדָה |

### 5.2 Правило: не все глаголы имеют שם פעולה в обиходе

Некоторые отглагольные существительные очень частотны (הוֹדָעָה — сообщение),
другие — редки или книжны (הוֹשָׁבָה — усаживание). Если в слайде
shem_peula перечислены конкретные формы — включать именно их.

### 5.3 Сопоставление с глаголами из предыдущих частей

Если в текущей части появляются שם פעולה от глаголов, введённых в предыдущих
частях, их всё равно нужно включить — это показывает студенту полную картину.

Дополнительно можно добавить שם פעולה от глаголов текущей части, которые
не были явно даны в slайде, но легко образуются по модели.

## 6. Обработка специальных грамматических тем

### 6.1 אוגד (связка)

Если в уроке есть слайды про אוגד:

1. Объяснить, что такое именное предложение (משפט שמני)
2. Показать нулевую связку в настоящем времени
3. Объяснить местоименную связку (הוא/היא/הם/הן + тире)
4. Привести примеры
5. Сравнить три типа связки: нулевая, местоименная, разговорная (זה)
6. Объяснить использование הָיָה для прошедшего/будущего
7. Добавить исторический экскурс (библейские примеры)

### 6.2 Конструкции противопоставления (לא...אלא, לא רק...אלא גם)

Если в уроке есть эти конструкции:
1. Объяснить структуру
2. Дать примеры из урока
3. Показать разницу между לא...אלא и לא רק...אלא גם

### 6.3 Числительные, предлоги, и т.д.

Если в уроке есть соответствующие слайды — обрабатывать по тому же принципу:
извлечь, структурировать, обогатить пояснениями, добавить огласовки.

## 7. Исторические и этимологические комментарии

### 7.1 Когда добавлять

- При объяснении нерегулярных или неожиданных форм (почему [o] в פ״י?)
- При объяснении грамматических явлений с глубокими корнями (אוגד в Танахе)
- Когда это помогает запомнить правило

### 7.2 Принципы

- Краткость: 1-3 абзаца максимум
- Связь с практикой: объяснение должно помочь понять или запомнить форму
- Упоминание параллелей в арабском / арамейском, если уместно
- Ссылки на библейские формы, если уместно
- **Каждый комментарий должен отвечать на конкретный вопрос**:
  «Почему в פ״י звук [o]?», «Откуда взялась связка הוא?»

**Антипаттерн (из ранних шпаргалок):**
НЕ писать многостраничные академические обзоры с периодизацией
(«Библейский иврит → Мишнаитский → Средневековый → Современный»).
Это перегружает справочник и не помогает студенту запомнить форму.

**Хороший пример:** «Звук [o] в הוֹרִיד — след древнего *haw-*, где w —
остаток первой корневой буквы» (3 предложения, привязан к конкретной форме).

**Плохой пример:** «В библейском иврите модальность выражалась через когортатив
и юссив. В мишнаитском периоде (200-500 гг.) под влиянием арамейского...»
(многоабзацный обзор, не привязанный к форме из урока).

### 7.3 Пример

```markdown
## 1.6 Исторический комментарий

Происхождение гласного [o] связано с древней формой каузативного префикса *haw-*.

Исторически: *hawrid → hōrīd*

Буква י в начале корня взаимодействовала с префиксом через промежуточный
полугласный *w*, который развился в долгий [ō] (холам).

Таким образом, звук [o] в הוֹרִיד — это «археологический след»
древнего сочетания *haw-*, где w — остаток от первой корневой буквы.

Тот же процесс произошёл с аналогичными корнями и в арабском языке
(четвёртая порода: *ʔawrada*).
```

## 9. Эталонный стиль и антипаттерны

Формат шпаргалок эволюционировал от урока к уроку. Текущий эталон —
шпаргалки урока 13 (части 1-3). Все новые шпаргалки должны следовать
этому стилю.

### 9.1 Что ДЕЛАТЬ (эталон — урок 13)

- Три колонки в словаре: **С огласовками | Без огласовок | Перевод**
- Символ **↳** для שם פעולה в сводных таблицах глаголов
- Краткие исторические комментарии (1-3 абзаца), привязанные к конкретной форме
- Таблицы для парадигм спряжений (строки = лица, столбцы = формы)
- Blockquote (`>`) для ключевых правил и формулировок
- Пары примеров: иврит первой строкой, русский под ним, без буллетов
- Нумерованные разделы (`# 1. ...`, `## 1.1 ...`)
- Диагностическая сводная таблица (как различать гзарот)

### 9.2 Чего НЕ делать (устаревшие паттерны ранних шпаргалок)

- ❌ **ASCII-диаграммы формул** в code-блоках (`מִתְ + פ + ַ + ע...`).
  Были в уроке 11, больше не используются. Вместо этого — обычные таблицы
  и текстовые списки с bold.

- ❌ **Многостраничные исторические разделы** с периодизацией
  (библейский → мишнаитский → средневековый → современный).
  Были в уроке 12. Вместо этого — краткий комментарий, связанный
  с конкретной формой.

- ❌ **Bold на иврите в словарных таблицах** (`**להתלבש**`).
  Был в уроке 11. В текущем формате иврит в таблицах идёт без bold.
  Bold допускается только для инфинитивов в сводной таблице глаголов
  с ↳ שם פעולה.

- ❌ **Колонка «Примечания»** в словаре (`Иврит | Перевод | Примечания`).
  Была в уроке 11. В текущем формате — три колонки без примечаний.
  Если нужно пояснение, оно идёт в скобках внутри перевода.

- ❌ **Тематическая группировка глаголов** (по темам: «эмоции»,
  «повседневные действия» и т.д.). Была в уроке 11. В текущем формате
  глаголы группируются только по гзарот.

- ❌ **Культурные заметки и экскурсы**, не связанные напрямую
  с грамматикой урока. Были в уроках 11-12.

### 9.3 Допустимые вариации

Некоторые элементы зависят от содержания конкретного урока:

- Если урок вводит новый биньян целиком (как урок 11 — התפעל, урок 13 — הפעיל),
  грамматический раздел будет объёмнее, чем при объяснении частного явления.
- Количество контрастных пар зависит от того, сколько их в исходных слайдах.
- Кумулятивная таблица глаголов появляется только если урок посвящён
  конкретному биньяну и разбит на несколько частей.
- Диагностическая памятка нужна только когда в уроке представлены
  несколько гзарот одного биньяна.

### 9.4 Объём и пропорции

Ориентировочные пропорции разделов шпаргалки:

| Раздел | Доля от общего объёма |
|---|---|
| Грамматика (объяснения + парадигмы + примеры) | 40-50% |
| Сводная таблица глаголов (кумулятивная) | 15-20% |
| Словарь новых слов | 15-20% |
| שם פעולה | 10-15% |
| Диагностика / памятки | 5-10% |
