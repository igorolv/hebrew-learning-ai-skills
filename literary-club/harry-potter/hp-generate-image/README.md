# hp-generate-image

Claude Skill для генерации готового промта для создания книжной иллюстрации к странице ивритского Гарри Поттера.

## Что делает

- Принимает фрагмент текста сцены и сопоставляет его с описанием из chapter style
- Извлекает визуальные детали персонажей, локации и атмосферу
- Формирует промт в стиле традиционной книжной иллюстрации (ink + watercolor, 1990s)
- Учитывает формат A4 (иллюстрация в верхней половине страницы)
- Включает строгие запреты (photorealism, магическое свечение, anime и т.д.)

## Как использовать

### В Claude.ai (Projects)
Добавьте файл `SKILL.md` в Project Knowledge вашего проекта.

### В Claude Code
Скопируйте папку в директорию skills и подключите через конфигурацию.

## Структура

```
hp-generate-image/
├── SKILL.md    — основная инструкция
└── README.md
```

## Входные данные

1. `master_style_framework.md` — глобальный стиль иллюстраций (приоритетный документ)
2. `BOOK_{N}_CHAPTER_{M}_STYLE.md` — описание персонажей, локаций и сцен главы (результат работы `hp-chapter-style`)
3. Фрагмент текста сцены на русском — конкретный момент для иллюстрации

## Выходные данные

Один готовый промт на английском языке для генератора изображений.
