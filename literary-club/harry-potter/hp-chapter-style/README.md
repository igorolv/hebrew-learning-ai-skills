# hp-chapter-style

Claude Skill для генерации поглавного иллюстрационного стайл-гайда по тексту Гарри Поттера.

## Что делает

- Анализирует текст главы (перевод Росмэн) и создаёт дельта-документ для иллюстраций
- Формирует эталоны персонажей (внешность, одежда, пластика) на основе текста
- Описывает локации с палитрой, освещением и уровнем магии
- Разбивает главу на визуальные сцены с уникальными ID
- Создаёт сводку главы для иллюстратора

## Как использовать

### В Claude.ai (Projects)
Добавьте файл `SKILL.md` в Project Knowledge вашего проекта.

### В Claude Code
Скопируйте папку в директорию skills и подключите через конфигурацию.

## Структура

```
hp-chapter-style/
├── SKILL.md                — основная инструкция
├── README.md
└── references/
    └── master_style_framework.md    — неизменяемый каркас стиля проекта
```

## Входные данные

Текст одной главы Гарри Поттера на русском языке (перевод издательства Росмэн).

## Выходной файл

`BOOK_{N}_CHAPTER_{M}_STYLE.md` — дельта-документ с персонажами, локациями и сценами главы.
