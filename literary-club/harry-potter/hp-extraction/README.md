# hp-extraction

Claude Skill для извлечения ивритского текста из PDF-исходника «Гарри Поттера» и сборки в структурированный markdown.

## Что делает

- Подготавливает изображения страниц из PDF или ZIP-архива (скрипт `prepare_pdf.py`)
- Визуально читает ивритский текст с изображений
- Сохраняет оригинальные огласовки (никуд) и форматирование
- Извлекает таблицы сложных слов
- Собирает всё в единый markdown-файл

## Как использовать

### В Claude.ai (Projects)
Добавьте файл `SKILL.md` в Project Knowledge вашего проекта.

### В Claude Code
Скопируйте папку в директорию skills и подключите через конфигурацию.

## Структура

```
hp-extraction/
├── SKILL.md                — основная инструкция
├── README.md
├── scripts/
│   └── prepare_pdf.py      — подготовка изображений из PDF/ZIP
└── references/
    └── HP_ch1_20_25.md     — образец результата (страницы 20–25)
```

## Зависимости

- Python 3
- `pymupdf` — устанавливается автоматически скриптом при первом запуске

## Пример задания

> «Файл Исходник Гарри Поттер 1 книга 1 стр 1 35.pdf, глава 1, обработай с 30 страницы по 35 страницу»
