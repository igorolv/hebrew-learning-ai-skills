# hp-master-style

Claude Skill для управления `master_style_framework.md` — центральным документом визуального стиля проекта иллюстрирования Гарри Поттера.

## Что делает

- Создаёт `master_style_framework.md` из дефолтного шаблона для нового проекта
- Валидирует существующий файл: проверяет наличие обязательных секций и консистентность
- Расширяет каркас при переходе к новой книге (добавляет секцию IX+ без изменения ядра)
- Показывает текущий master style из Project Knowledge

## Зачем отдельный скилл

`master_style_framework.md` — единственный файл, который используют сразу два скилла:
- `hp-chapter-style` (читает при создании поглавного стайл-гайда)
- `hp-generate-image` (читает при генерации промта для иллюстрации)

Отдельный скилл даёт одну точку ответственности за создание, валидацию и обновление этого файла.

## Как использовать

### В Claude.ai (Projects)
Добавьте файл `SKILL.md` в Project Knowledge вашего проекта.

### Команды
- **«Создай master style»** — генерирует файл из шаблона, пользователь добавляет в PK
- **«Проверь master style»** — валидация секций и консистентности
- **«Расширь master style для книги N»** — добавляет дополнения без изменения ядра
- **«Покажи master style»** — находит и отображает текущую версию из PK

## Часть проекта

Этот skill — **нулевой этап** (setup), предшествующий пайплайну иллюстрирования:

0. **hp-master-style** ← вы здесь (создаёт каркас стиля)
1. [hp-chapter-style](../hp-chapter-style) — поглавный стайл-гайд (использует каркас)
2. [hp-generate-image](../hp-generate-image) — промт для иллюстрации (использует каркас)
