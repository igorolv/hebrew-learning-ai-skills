# hebrew-hp-source-texts

Claude Skill для поиска английского оригинала и русского перевода (Росмэн) нужной главы Гарри Поттера в Project Knowledge.

## Что делает

- Ищет файлы исходных текстов в Project Knowledge по конвенции имён
- Извлекает фрагмент, соответствующий обрабатываемым страницам ивритской книги
- Если файлы не найдены — запрашивает у пользователя
- Предоставляет английский и русский тексты для использования другими skill'ами

## Конвенция имён

Файлы в Project Knowledge:
- `HP_en_ch{N}.md` — английский оригинал главы N
- `HP_rosmen_ch{N}.md` — русский перевод Росмэн главы N

## Как использовать

### В Claude.ai (Projects)
1. Добавьте `SKILL.md` в Project Knowledge
2. Загрузите файлы исходных текстов (`HP_en_ch1.md`, `HP_rosmen_ch1.md` и т.д.)

### В Claude Code
Скопируйте папку в директорию skills и подключите через конфигурацию.

## Структура

```
hebrew-hp-source-texts/
├── SKILL.md    — основная инструкция
└── README.md
```
