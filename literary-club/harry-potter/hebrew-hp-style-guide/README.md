# hebrew-hp-style-guide

Claude Skill для пополнения HEBREW_HARRY_POTTER_MASTER_STYLE — гайда по визуальному стилю для иллюстраций к ивритскому Гарри Поттеру.

## Что делает

- Анализирует новый фрагмент текста на наличие новых персонажей, локаций и сцен
- Добавляет эталоны новых персонажей (секция IV)
- Добавляет новые локации с палитрой и уровнем магии (секция IX)
- Документирует сцены с отличиями от эталона (секция X)
- Обновляет лог обработки (секция XI)
- Не трогает зафиксированные разделы стиля (I–III, V–VIII)

## Как использовать

### В Claude.ai (Projects)
Добавьте файл `SKILL.md` в Project Knowledge вашего проекта.

### В Claude Code
Скопируйте папку в директорию skills и подключите через конфигурацию.

## Структура

```
hebrew-hp-style-guide/

├── README.md
└── references/
    └── HEBREW_HARRY_POTTER_MASTER_STYLE_example.md   — образец заполненного стайлгайда
```

## Входные данные

- Новый фрагмент текста (страницы N–M)
- Текущая версия `HEBREW_HARRY_POTTER_MASTER_STYLE.md`
