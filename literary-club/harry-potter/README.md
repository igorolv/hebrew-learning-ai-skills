# Harry Potter · Hebrew Learning Pipeline

Набор Claude Skills для создания учебных материалов по ивритскому изданию «Гарри Поттера» — от сырого PDF до готового DOCX с иллюстрациями.

---

## Начальные условия

Перед началом работы в Project Knowledge должны быть размещены:

### Обязательно

1. **Все SKILL.md файлы** из папок скилов (или ссылки на них в Claude Code)
2. **PDF-исходник** ивритской книги (файл вида `Исходник_Гарри_Поттер_книга_N_глава_M.pdf`) — фактически ZIP-архив с JPEG-изображениями и текстовыми файлами страниц
3. **Английский оригинал главы** — файл `HP_en_ch{N}.md` (полный текст главы, plain text в markdown)
4. **Русский перевод Росмэн** — файл `HP_rosmen_ch{N}.md` (полный текст главы по изданию Росмэн)

### Для иллюстрационного пайплайна

5. **master_style_framework.md** — неизменяемый каркас визуального стиля всей книги (лежит в `hp-chapter-style/references/`, нужен для chapter style и генерации промтов)
6. **Текст главы на русском** (Росмэн) — тот же `HP_rosmen_ch{N}.md`, но используется целиком, не фрагментами

### Рекомендуется

7. **Результаты предыдущих обработок** той же главы (если есть) — для сверки формата и преемственности

## Два пайплайна

Скилы образуют два независимых пайплайна, которые можно вести параллельно.

### Пайплайн A — Учебные материалы (текст → DOCX)

```
PDF-исходник (ZIP с JPEG)
        │
        ▼
  ┌──────────────┐
  │ hp-extraction │   Извлечение ивритского текста
  └──────┬───────┘
         │  → HP_ch{N}_{FROM}_{TO}.md
         ▼
  ┌──────────────┐       ┌─────────────────┐
  │ hp-translate  │ ◄──── │ hp-source-texts  │  (утилита: ищет EN + Росмэн)
  └──────┬───────┘       └─────────────────┘
         │  → HP_ch{N}_{FROM}_{TO}_translate.md
         ▼
  ┌──────────────────┐
  │ hp-generate-docx  │   + ZIP с иллюстрациями
  └──────┬───────────┘
         │
         ▼
  Гарри Поттер глава {N} страницы {FROM}-{TO}.docx
```

### Пайплайн B — Иллюстрации (текст → промт)

```
Текст главы (Росмэн, целиком)
        │
        ▼
  ┌──────────────────┐
  │ hp-chapter-style  │   + master_style_framework.md
  └──────┬───────────┘
         │  → BOOK_{N}_CHAPTER_{M}_STYLE.md
         ▼
  ┌──────────────────┐
  │ hp-generate-image │   + фрагмент текста конкретной сцены
  └──────┬───────────┘
         │
         ▼
  Промт для генератора изображений (EN)
```

## Подробно по шагам

### Шаг 1 · hp-extraction

**Что делает:** распаковывает PDF (ZIP-архив), визуально читает ивритский текст с изображений, извлекает таблицы сложных слов.

| | |
|---|---|
| **Вход** | PDF-исходник + номер главы + диапазон страниц |
| **Выход** | `HP_ch{N}_{FROM}_{TO}.md` — markdown с ивритским текстом и таблицами слов |
| **Пример** | `HP_ch1_30_35.md` |

**Что проверять:**
- Ивритский текст совпадает с изображением (особенно огласовки на именах собственных)
- Таблица сложных слов полная — все слова с картинки присутствуют
- Жирный шрифт сохранён
- Кавычки ивритские ״...״
- Нет лишних или пропущенных абзацев

---

### Шаг 2 · hp-translate

**Что делает:** добавляет селективные огласовки (никуд), подстрочный перевод, литературный перевод Росмэн, анализ различий.

**Зависимость:** автоматически вызывает `hp-source-texts` для поиска `HP_en_ch{N}.md` и `HP_rosmen_ch{N}.md` в Project Knowledge.

| | |
|---|---|
| **Вход** | `HP_ch{N}_{FROM}_{TO}.md` (результат шага 1) |
| **Выход** | `HP_ch{N}_{FROM}_{TO}_translate.md` — полный учебный markdown |
| **Пример** | `HP_ch1_30_35_translate.md` |

**Что проверять:**
- Огласовки (никуд): «матери чтения» (вав, йуд) не удалены — сохранено полное написание (ктив мале)
- Простые слова (אני, של, את и т.д.) НЕ огласованы, сложные — огласованы
- Имена собственные всегда с огласовками
- Подстрочный перевод: фрагменты в таблице совпадают с текстом в секции «Иврит»
- Литературный перевод: точный текст Росмэн, не пересказ
- Различия переводов: 3–5 значимых наблюдений на страницу, не формальные мелочи

---

### Шаг 3 · hp-generate-docx

**Что делает:** преобразует учебный markdown в форматированный DOCX с иллюстрациями.

| | |
|---|---|
| **Вход** | `HP_ch{N}_{FROM}_{TO}_translate.md` (результат шага 2) + ZIP-архив с иллюстрациями |
| **Выход** | `Гарри Поттер глава {N} страницы {FROM}-{TO}.docx` |

**Что проверять:**
- Каждая страница начинается с нового листа Word
- Иллюстрация присутствует на каждой странице
- Иврит отображается шрифтом David, 18pt, выравнивание вправо
- Таблицы не разорваны между страницами
- Заголовки секций (## Иврит, ## Подстрочный перевод и т.д.) удалены — остался только «Страница N»

---

### Шаг 4 · hp-chapter-style

**Что делает:** анализирует полный текст главы (Росмэн) и создаёт дельта-документ для иллюстраций — персонажи, локации, сцены с уникальными ID.

| | |
|---|---|
| **Вход** | Полный текст главы на русском (Росмэн) + номер книги и главы |
| **Выход** | `BOOK_{N}_CHAPTER_{M}_STYLE.md` |
| **Пример** | `BOOK_1_CHAPTER_1_STYLE.md` |

**Что проверять:**
- Все персонажи из главы учтены (впервые появляющиеся — с полным эталоном, упомянутые — с заглушкой)
- Визуальные описания основаны на тексте, а не на фильмах
- Метафоры переведены в конкретные визуальные инструкции
- Уровни магии (1/2/3) проставлены для каждой сцены и локации
- Сцены достаточно гранулярные — каждая смена места, персонажей или настроения = отдельная сцена
- Эмоциональный тон описан как динамика, не статика

---

### Шаг 5 · hp-generate-image

**Что делает:** генерирует готовый промт на английском для создания книжной иллюстрации.

| | |
|---|---|
| **Вход** | `master_style_framework.md` + `BOOK_{N}_CHAPTER_{M}_STYLE.md` + фрагмент текста сцены |
| **Выход** | Текстовый промт на английском языке |

**Важно:** генерация иллюстрации — итеративный процесс. Рекомендуется вести его в **отдельном чате**, чтобы можно было корректировать промт, уточнять детали, менять ракурс или композицию, не засоряя основной рабочий чат. Промт — это стартовая точка, а не финальный результат. Может потребоваться несколько итераций, чтобы добиться нужной картинки.

**Что проверять в промте:**
- Присутствует обязательное начало (Book illustration for the TOP HALF of a vertical A4 page...)
- Персонажи соответствуют эталонам из chapter style
- Нет запрещённых элементов (photorealism, glowing effects, anime и т.д.)
- Стиль — traditional ink and watercolor, не digital painting

## Утилиты

### hp-source-texts

Вспомогательный скил, который не вызывается напрямую. Используется скилом `hp-translate` для поиска английского оригинала и русского перевода Росмэн в Project Knowledge.

Ищет файлы по конвенции имён:
- `HP_en_ch{N}.md` — английский оригинал
- `HP_rosmen_ch{N}.md` — русский перевод Росмэн

Если файлы не найдены — запрашивает у пользователя.

## Конвенции именования файлов

| Файл | Формат имени | Пример |
|---|---|---|
| Исходник (PDF/ZIP) | `Исходник_Гарри_Поттер_книга_{N}_глава_{M}.pdf` | `Исходник_Гарри_Поттер_книга_1_глава_1.pdf` |
| Английский оригинал | `HP_en_ch{N}.md` | `HP_en_ch1.md` |
| Русский Росмэн | `HP_rosmen_ch{N}.md` | `HP_rosmen_ch1.md` |
| Извлечённый иврит | `HP_ch{N}_{FROM}_{TO}.md` | `HP_ch1_30_35.md` |
| Переведённый файл | `HP_ch{N}_{FROM}_{TO}_translate.md` | `HP_ch1_30_35_translate.md` |
| Готовый DOCX | `Гарри Поттер глава {N} страницы {FROM}-{TO}.docx` | `Гарри Поттер глава 1 страницы 30-35.docx` |
| Chapter style | `BOOK_{N}_CHAPTER_{M}_STYLE.md` | `BOOK_1_CHAPTER_1_STYLE.md` |
| Master style | `master_style_framework.md` | — |
