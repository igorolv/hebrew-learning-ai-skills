# AGENTS.md

Общие инструкции для любого AI-агента, работающего с этим репозиторием.

## Что это

Репозиторий skill'ов для изучения иврита. Skill — это самодостаточная инструкция для AI-агента, описывающая один шаг рабочего процесса. Skill'ы организованы в pipeline'ы — цепочки последовательных шагов.

## Направления

### 1. Homework (`homework/`)

Обработка уроков иврита и выполнение домашних заданий.

Граф скилов, порядок запуска, стыки между шагами — в [`homework/PIPELINE.md`](homework/PIPELINE.md).

| Skill | Что делает |
|---|---|
| `hebrew-lesson-extract` | Извлечение содержимого PPTX-урока в markdown |
| `hebrew-lesson-enrich` | Грамматическая шпаргалка по extracted-файлу |
| `hebrew-homework-solve` | Выполнение упражнений по выбранным слайдам |
| `hebrew-homework-docx` | Конвертация ДЗ в DOCX |
| `hebrew-lesson-docx` | Конвертация шпаргалки в DOCX |

### 2. Literary Club (`literary-club/harry-potter/`)

Учебные материалы по ивритскому изданию «Гарри Поттера».

Граф скилов (текстовая + иллюстрационная ветки), порядок запуска, стыки — в [`literary-club/harry-potter/PIPELINE.md`](literary-club/harry-potter/PIPELINE.md).

| Skill | Что делает |
|---|---|
| `hp-extraction` | Извлечение ивритского текста из PDF (ZIP с JPEG) |
| `hp-source-texts` | Поиск английского оригинала и перевода Росмэн |
| `hp-translate` | Огласовки, подстрочный и литературный перевод |
| `hp-master-style` | Master style framework для иллюстраций |
| `hp-chapter-style` | Поглавный визуальный стайлгайд |
| `hp-generate-image` | Промты для иллюстраций |
| `hp-generate-docx` | Сборка финального DOCX |

## Структура skill'а

Каждый skill — это папка, содержащая:

- `SKILL.md` — основная инструкция (с YAML frontmatter: `name`, `description`)
- `README.md` — краткое описание для человека
- `references/` — образцы, шаблоны, вспомогательные файлы

## Языки

- **Иврит** — целевой язык изучения. Огласовки (никуд) критически важны и должны точно соответствовать источнику.
- **Русский** — рабочий язык инструкций, переводов, комментариев, общения с пользователем.
- **Английский** — используется только для промтов генерации иллюстраций и технических терминов.

## Принципы работы

- Skill'ы выполняются строго по инструкции из `SKILL.md`. Не импровизировать.
- Файлы `references/` содержат образцы формата — следовать им точно.
- Не добавлять и не убирать огласовки (никуд) самовольно — только как указано в SKILL.md.
- Имена собственные на иврите всегда с огласовками.
- Конвенции именования файлов зафиксированы в README.md каждого pipeline'а — соблюдать их.
- Выходные файлы создавать в рабочей директории пользователя, не в репозитории skill'ов.
